## Aesthetic Customisations

Many of the icons and control colors can be modified.

## Upload Destinations

Images captured by CaptureThat can be uploaded to the following destinations:

* [Gyazo](http://gyazo.com)
* [Imgur](http://imgur.com)
* [vgy.me](http://vgy.me)
